== Description ==

Adding a custom field to the post, it can indicate that post is original or no.
Then add custom copyright declaration under those original posts.
增加一个自定义域可以设置文章是否为原创，然后可以为原创文章显示定制的版权声明。

详情：[https://s3.amazonaws.com/bbwordpress/index.html](https://s3.amazonaws.com/bbwordpress/index.html)
== Installation ==

可以通过以下两种方法的其中一种来安装Copyright Switch插件：

1. 将下载的文件解压缩，然后将‘copyright-switch’文件夹 上传到 `/wp-content/plugins/`目录，在插件后台启用

2. 直接在后台-安装插件，搜索'Copyright Switch'，按照提示安装启用
